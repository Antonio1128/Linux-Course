# Linux Bash Scripts

This repository contains bash scripts I created while learning Linux.

## Scripts

system_info.sh - Displays basic system information (CPU, memory, disk usage).  
backup.sh - Creates backups of files or directories.  
auto_commit.sh - Automates git add, commit, and push.    
file_counter.sh - Counts files in a directory.  
file_finder_tool.sh - Searches for files by name or type  
git_clone_project.sh - Clones a Git repository.  
git_repinfo.sh - Shows repository details (branch, status, remote).  
log_analyzer.sh - Analyzes logs and extracts useful info.  
project_setup.sh  - Generates a basic project structure.  
website_monitor.sh - Checks if websites are online.  
password_generator.sh - Generates random secure passwords.  

## Mini Projects  

cleanup_tool.sh - Cleans system junk and cache.   
file_organizer_tool.sh - Organizes files into folders by type.  

### How to run?  

 chmod +x script_name.sh     
 ./script_name.sh
