# Linux Mini Projects

This folder contains small Linux tools and utilities written in Bash.

## Projects

### File Organizer

A tool that organizes files in a directory by file type.

Usage:

./organize.sh <directory>

Example:

./organize.sh ~/Downloads

### Linux Cleanup Tool

A Bash tool that cleans temporary files and cache from a Linux system.

Features

* cleans /tmp
* cleans user cache
* empties trash
* dry run mode

Usage

./cleanup_tool.sh

Dry run:
/cleanup_tool.sh -- dry-run
./cleanup.sh --dry-run
```
